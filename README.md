# Data-Cleaning-Transformation
Data cleaning and transforming using Excel

📌 Overview

This project focuses on **data cleaning and preprocessing** to prepare raw data for accurate analysis and reporting. The objective was to improve data quality, consistency, and reliability before performing Exploratory Data Analysis (EDA).

🛠 Cleaning Steps Performed

* Removed null and blank values
* Eliminated duplicate records
* Converted text-type numeric columns to number format
* Standardized column names and formatting
* Split and merged columns where necessary
* Handled inconsistent categorical values
* Validated data types and relationships

🔍 Tools Used

* **Excel** – Basic cleaning, sorting, filtering
* **Power Query** – Data transformation and automation
* **Power BI** – Data validation and modeling

📈 Outcome

* Improved data accuracy and consistency
* Reduced errors in analysis and dashboards
* Created a clean, structured dataset ready for reporting
